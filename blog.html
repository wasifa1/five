Question-1: What are the different ways to select an element in the DOM?

1. getElementById(id) : Selection through ID
2. getElementsByClassName(class) : Selection through class name
3. getElementsByTagName(tag) : Selection through tag
4. querySelector(selector) : Selection of first element fulfilling the condition
5. querySelectorAll(selector) : Selection of all elements fulfilling the condition

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML : Sets HTML code inside an element
2. innerText : Sets the text inside an element without showing HTML tags
3. textContent : Sets text along with hidden elements

Question-3: What is event delegation in the DOM?

The process of using Event Listener on parent instead of using it separately for each child is Event Delegation.

Question-4: What is event bubbling in the DOM?

The process in which events are handled in three steps :
1. Capture Phase : From parent to target element
2. Target Phase : Selecting the target element
3. Bubbling Phase : Goes back from target element to parent

Question-5: How do you create, add, and remove elements using JavaScript?

The processes are given below :
1. Create : document.createELement(tag)
2. Add : parentElement.appendChild(element)
3. Remove : parentElement.removeChild(element)
